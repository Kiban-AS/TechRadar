[Grafana](https://grafana.com/) is an open-source analytics and visualization platform used for monitoring, observability, and operational intelligence. It connects to a wide range of data sources including Prometheus, Loki, Elasticsearch, and Azure Monitor, enabling teams to build rich dashboards and alerting rules for production workloads. Grafana is a core component of the observability stack used in Kiban's platform engineering work, providing actionable metrics, logs, and dashboards for Kubernetes clusters and platform services. It has been used to create custom metrics dashboards, SLA-aligned alerting, and on-call notification workflows. Grafana is the recommended visualization layer for any team operating production Kubernetes infrastructure.
