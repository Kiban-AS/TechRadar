[FluxCD](https://fluxcd.io/) is a set of continuous and progressive delivery solutions for Kubernetes that are open and extensible. It enables GitOps workflows by continuously reconciling cluster state with configuration stored in Git repositories. FluxCD is a core tool in the Kiban platform engineering stack, used to manage automated namespace provisioning, application deployments, Helm releases, and image update automation across development and production environments. It supports safe promotion of changes across environments with minimal manual intervention and integrates tightly with Flux Operator for improved operational control and safer reconciliation.
