[Terraform](https://www.terraform.io/) is an open-source Infrastructure as Code tool developed by HashiCorp that enables teams to define and provision infrastructure using a declarative configuration language. Terraform supports a wide range of cloud providers including Azure, AWS, and GCP, and is widely used for provisioning and managing cloud resources in a repeatable and auditable way. It has been used extensively to automate Azure delivery workflows, provision Azure DevOps pipelines, and bootstrap infrastructure across environments in a standardized and traceable manner. Terraform is a proven and stable tool recommended for any team building platform engineering or cloud automation workflows.
