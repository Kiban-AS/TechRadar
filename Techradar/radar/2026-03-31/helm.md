[Helm](https://helm.sh/) is the package manager for Kubernetes, used to define, install, and upgrade complex Kubernetes applications. Helm charts provide a standardized and reusable way to package application deployments, enforce security and policy defaults, and improve developer productivity. Helm is used as the standard for both application deployments and platform tooling, with charts managed through a Helm OCI repository in JFrog Artifactory for consistent consumption across GitLab runners, VMs, cloud infrastructure, and AKS clusters. Helm is a fundamental tool in the Kubernetes ecosystem and is strongly recommended for any team managing application delivery on Kubernetes.
