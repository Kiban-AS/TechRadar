[Azure CLI](https://learn.microsoft.com/en-us/cli/azure/) is a cross-platform command-line tool for managing Azure resources and services. It provides a consistent scripting interface for provisioning, configuring, and querying Azure resources, and is especially useful in CI/CD pipelines, automation scripts, and GitOps workflows where Terraform or Bicep may not cover all operations. The Azure CLI has been used for automating namespace provisioning, secret rotation for FluxCD sync tokens, and configuration of AKS clusters and associated Azure resources. It is the essential companion tool to Bicep and Terraform for any team working on Azure-based platform engineering.
