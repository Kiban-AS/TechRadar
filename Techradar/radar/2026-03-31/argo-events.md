[Argo Events](https://argoproj.github.io/argo-events/) is an event-driven workflow automation framework for Kubernetes that enables triggering Argo Workflows, Lambda functions, and other actions in response to events from a wide range of sources — including webhooks, S3, SQS, Kafka, and Azure Storage Queue. It was introduced alongside Argo Workflows to drive event-based cluster automation and decouple event sources from workflow execution. While still being assessed at scale, it shows clear value for teams wanting to build reactive, event-driven platform automation on top of Kubernetes.
