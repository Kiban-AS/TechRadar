[Renovate Bot](https://docs.renovatebot.com/) is an automated dependency update tool that keeps software dependencies current through automated pull requests. It supports a wide range of package managers, container images, Helm charts, Terraform modules, and Kubernetes manifests. Renovate has been deployed to keep platform dependencies up to date, including proactive updates for deprecated Kubernetes API versions and Azure Verified Modules (AVM) using custom data sources. It reduces the risk of dependency drift and security vulnerabilities from outdated packages, and is a strong complement to any GitOps-based delivery workflow.
