[Amazon Elastic Kubernetes Service (EKS)](https://aws.amazon.com/eks/) is AWS's managed Kubernetes service that simplifies running Kubernetes clusters without needing to manage the control plane. EKS integrates natively with AWS services including IAM, VPC, ECR, CloudWatch, and ALB, and supports both EC2 and Fargate node types. EKS has been used to deploy and manage containerized microservices, maintain cluster health, deploy applications with Helm, and manage team access controls. Teams already running on AWS infrastructure should evaluate EKS as their Kubernetes runtime — though teams on Azure will find AKS a more natural fit with tighter native integrations.
