[SQL (Structured Query Language)](https://en.wikipedia.org/wiki/SQL) is the standard language for managing and querying relational databases, supported by virtually all relational database management systems (RDBMS) including PostgreSQL, MySQL, Oracle DB, and SQL Server. SQL skills are fundamental for platform engineers and DevSecOps practitioners who need to query monitoring databases, build operational dashboards, maintain operational data stores, and diagnose application issues at the data layer. SQL has been used for creating database views to surface errors and critical events, querying operational metrics, and supporting application development and operations. SQL is a core competency that all platform and operations engineers should maintain.
