[PowerShell](https://learn.microsoft.com/en-us/powershell/) is a cross-platform task automation and configuration management framework consisting of a command-line shell and scripting language built on .NET. Originally a Windows-only tool, PowerShell Core is now available on Linux and macOS, making it increasingly relevant in cross-platform DevOps workflows. PowerShell is used for scripting and automation tasks on Windows systems, Azure resource management, and integration with Microsoft tooling. It is particularly useful in environments with significant Microsoft infrastructure, where Azure CLI and PowerShell modules provide the most complete API coverage. Teams working in mixed Windows/Linux environments should evaluate PowerShell alongside Bash for automation tasks.
