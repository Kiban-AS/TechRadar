[GitHub Actions](https://github.com/features/actions) is GitHub's built-in CI/CD and automation platform that enables teams to automate software build, test, and deployment workflows directly from their GitHub repositories. It uses YAML-based workflow files stored in the repository, supporting reusable workflows, composite actions, and a large marketplace of community-built actions. GitHub Actions has been used to create and maintain automation for service enrollment and configuration, as well as static site deployments. Teams already using GitHub for source control will find GitHub Actions the most natural CI/CD choice, with tight integration across pull requests, environments, and secrets management.
