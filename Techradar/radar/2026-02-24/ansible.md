[Ansible](https://www.ansible.com/) is an open-source IT automation tool that enables configuration management, application deployment, and task automation using simple YAML-based playbooks. Unlike Terraform, Ansible focuses on configuration and orchestration of existing infrastructure rather than provisioning, making it a strong complement to IaC tools. Ansible has been used to configure servers and automate service enrollment using custom playbooks and GitHub Actions runners. It integrates well with existing infrastructure inventories and requires no agent on managed nodes, making it operationally lightweight. Teams should evaluate Ansible for configuration management tasks that go beyond what Terraform or Bicep handle natively.
